## To-Do List!

To-do list aplication, built with **React JS**.

## Features:

- #### Add a task with description and date
- #### Update a task
- #### Delete a task
- #### Hide/Show if a task is completed

## Technologies used:

- #### **React JS**
- #### **Tailwind CSS**

## Author:

- **Juan Quintero** - **[Portfolio Website](https://juanquintero.me)**

## Available Scripts:

#### `yarn install`

#### `yarn run dev`
